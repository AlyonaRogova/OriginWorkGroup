# Инструкция по работе с Git

## Инициализация репозитория

Для инициализации репозитория введите команду
''
git init
''

## Добавление файла в комид

Для добавления файла в комид введите команду
''
git add <file name>
''

## Создание коммита

Для создания нового коммита в текущей ветке введите команду

''
git commit -m "Commit message"
''

## Создание новой ветки

Для создания новой ветки введите команду
''
git branch <имя ветки>
''

## Слияние веток

Для слияния данной ветки с текущей введите команду
''
git merge <имя ветки>
''
## Перенос ветки на другой коммит

Для переноса ветки на вершину данной ветки введите команду
''
git rebase <имя ветки>
''

# Работа с удаленным репозиторием

## Клонирование репозитория на локальный ПК

Для переноса репозитория на свой ПК введите команду
''
git clone <url-адрес репозитория>
''

## Получение изменений и слияние с локальной версией

Для слияния с текущей версией на ПК введите команду
''
git pull
''

## Перемещение локального репозитория на внешний

Для переноса текущего репозитория на внешний введите команду
''git push
''
